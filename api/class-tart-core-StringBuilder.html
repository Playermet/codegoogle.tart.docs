def insert(insertPos:int32, src:String, start:int32, count:int32)

Append all or part of a String to the buffer at position insertPos.

Parameters
insertPos - Where to insert the characters.
src - The source String.
start - The start position within the source String. This will be clamped to the end of the source string.
count - The number of characters to insert. This will be clamped to the number of characters actually available.
Exceptions
IndexOutOfRangeException - If count or start are less than zero, or if the insertPos is outside of the range of the buffer.
