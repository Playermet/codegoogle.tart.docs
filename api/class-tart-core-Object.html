protected static def __flexAlloc[T](count:int32) -> T

Compiler intrinsic used to create and initialize instances of variable-length object types.

Type T represents a user-defined type whose last data member is a FlexibleArray. count represents the number of elements in the FlexibleArray.

__flexAlloc allocates a block of memory large enough to hold the object, using the default allocator, then initializes its TIB pointer to the TIB for T.
