alloc(size:uint32) -> Address[uint8]
Allocate size bytes of memory and return it. The memory will not be initialized, and may contain random data. If the allocator is one that supports garbage collection, it must fill in any fields that it needs, such as the __gcstate field of class Object.

free(ptr:Address[uint8])
