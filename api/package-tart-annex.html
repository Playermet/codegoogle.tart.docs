Package: tart.annex
Attributes
  • Coalesce - Attribute on a template which causes the compiler to merge instances of that template if possible. For example, if a template Foo is marked with this attribute, then the generated code for Foo[String] and Foo[Object] will be merged into a single definition.
  • Essential - Attribute that indicates that the associated definition is treated specially by the compiler. The evaluation of this attribute causes the associated definition to be registered by the compiler into a table of definitions that require special treatment.
  • GenerateStackTrace - Attribute used to indicate that an enumeration is a set of flags.
  • Intrinsic - Attribute that indicates that the associated function is a built-in compiler intrinsic.
  • TargetProperty - Attribute that indicates that this property is a global variable whose value is determined at link time based on the selection of the target platform.
